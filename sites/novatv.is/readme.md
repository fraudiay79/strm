# novatv.is

https://www.novatv.is/page/dagskra

### Download the guide

```sh
npm run grab -- --site=novatv.is
```

### Update channel list

```sh
npm run channels:parse -- --config=./sites/novatv.is/novatv.is.config.js --output=./sites/novatv.is/novatv.is.channels.xml
```

### Test

```sh
npm test -- novatv.is
```
