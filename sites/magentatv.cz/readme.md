# magentatv.cz

https://tvgo.t-mobile.cz/

### Download the guide

```sh
npm run grab --- --site=magentatv.cz
```

### Update channel list

```sh
npm run channels:parse --- --config=./sites/magentatv.cz/magentatv.cz.config.js --output=./sites/magentatv.cz/magentatv.cz.channels.xml
```

### Test

```sh
npm test --- magentatv.cz
```
